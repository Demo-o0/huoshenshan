火神山项目，Wuahan2020社区基础事务server端 Java 项目，jdk版本：java14 mysql8

## 快速启动
* 1、本地安装jdk、mysql server
* 2、初始化sql `/src/main/resources/sql`
* 3、在IDE 启动`src/main/java/org/wuhan2020/huoshenshan/HuoshenshanApplication.java` 主类
